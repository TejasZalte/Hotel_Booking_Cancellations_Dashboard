 🏨 Hotel Booking Cancellation Analysis | Microsoft Excel

 📌 Project Overview

This project focuses on analyzing hotel booking and cancellation patterns using Microsoft Excel. The dashboard provides actionable insights into customer behavior, booking trends, cancellation rates, market segments, and factors influencing hotel reservation cancellations. The objective is to help hotel management reduce cancellations, improve occupancy rates, and optimize revenue generation.

 🎯 Objectives

* Analyze hotel booking and cancellation trends.
* Identify key factors contributing to booking cancellations.
* Understand customer booking behavior.
* Explore seasonal patterns in reservations.
* Develop an interactive dashboard for business decision-making.

 📊 Key Insights & Features

# Booking Analysis

* Total Bookings and Confirmed Reservations.
* Total Cancelled Bookings.
* Booking status distribution.
* Monthly and yearly booking trends.

# Cancellation Analysis

* Overall cancellation rate.
* Cancellation trends across hotel types.
* Impact of lead time on cancellations.
* Cancellation behavior by customer segment.

# Customer Analysis

* Customer type distribution.
* Repeat vs New Guest analysis.
* Market segment performance.
* Special request trends and their impact on cancellations.

# Revenue & Occupancy Insights

* Booking patterns affecting occupancy rates.
* Potential revenue loss due to cancellations.
* Seasonal demand fluctuations.
* Peak booking periods identification.

# Interactive Dashboard Features

* Dynamic filters and slicers.
* Pivot Tables and Pivot Charts.
* KPI Cards for key metrics.
* Interactive trend analysis.
* User-friendly dashboard navigation.
